◆ SetSDCheck()

void comptonsoft::VCSSensitiveDetector::SetSDCheck ( bool  v = true)
inline

If you call this method with true, this class prints a warning message when an unregistered detector detects any hits. This setting is useful for detecting an incomplete registration of the sensitive detectors. With the parameter of false, this function is disabled, and such unregistered detectors are ignored. If you have any passive volumes associated with this sensitive logical volume, you should disable this function.
