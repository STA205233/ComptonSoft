Detailed Description

A class for simulating a semiconductor detector device. This class provides functions to convert a deposited energy of a hit into a detector output such as an electrical pulse height, taking into account of a detector response. Since the class also provides interface to such functions, you can derive a new specific class from this class to realize special treatment of detector response.
